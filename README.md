# Real-Time-AI-Captioning-System-using-Speech-and-Facial-Movement-Analysis

## 📌 Description

This project presents a real-time AI-based captioning system that generates subtitles from video input by combining audio processing and facial movement analysis. The primary goal is to ensure accurate caption generation even in challenging conditions such as missing, unclear, or noisy audio.

The system first attempts to extract and process speech from the audio using speech recognition techniques. However, when audio is unavailable or unreliable, it intelligently switches to a lip-reading approach by analyzing facial movements, particularly the speaker’s lip patterns, to predict and generate captions.

By integrating both audio and visual modalities, the model improves robustness and reliability in real-world scenarios such as video conferencing, surveillance, accessibility tools for the hearing impaired, and multimedia content processing.

Overall, this project demonstrates how multimodal AI systems can enhance captioning performance by overcoming limitations of single-source input methods.
